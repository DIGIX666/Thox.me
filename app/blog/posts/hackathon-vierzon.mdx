### How NFT-Invoices Work

The core of our innovation lies in transforming traditional invoices into NFT tokens:

```solidity
// Simplified structure of our NFT-Invoice
struct Invoice {
    uint256 id;
    address issuer;
    address client;
    uint256 amount;
    string description;
    InvoiceStatus status;
    uint256 timestamp;
}

enum InvoiceStatus {
    ISSUED,
    PAID,
    CANCELLED
}
```

This approach guarantees:
- **Uniqueness**: Each invoice is a unique token
- **Traceability**: Immutable transaction history
- **Verification**: Automatic payment validation

### Smart Payment Management

Our system automates the complete lifecycle of an invoice:

```solidity
function processPayment(uint256 invoiceId) external payable {
    Invoice storage invoice = invoices[invoiceId];
    require(invoice.status == InvoiceStatus.ISSUED, "Invoice already paid");
    require(msg.value >= invoice.amount, "Insufficient amount");

    // Transfer to supplier account
    payable(invoice.issuer).transfer(invoice.amount);

    // Status update
    invoice.status = InvoiceStatus.PAID;

    emit InvoicePaid(invoiceId, msg.sender, invoice.amount);
}
```

**Technical advantages:**
- **Atomicity**: Payment and update in one transaction
- **Security**: Automatic amount verification
- **Transparency**: Traceable blockchain events
